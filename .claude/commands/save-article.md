# /save-article

URL에서 아티클을 저장하고 메타데이터(요약, 키워드, 인사이트)를 자동 생성합니다.

## 사용법

```
/save-article <URL> [카테고리1, 카테고리2, ...]
```

## 워크플로우

1. **URL 확인**: 사용자가 제공한 URL 확인
2. **카테고리 확인**: 카테고리가 없으면 사용자에게 질문
3. **콘텐츠 읽기**: WebFetch로 URL 내용 확인
4. **메타데이터 생성**:
   - `summary`: 3-5문장 핵심 요약 (한국어)
   - `keywords`: 쉼표 구분 키워드 5-10개
   - `insights`: 핵심 인사이트 3-5개 (줄바꿈 구분)
5. **저장**: `save_article` MCP tool 호출
6. **결과 보고**: 저장된 아티클 정보 표시

## 실행

$ARGUMENTS가 비어있으면 URL을 요청하세요.

$ARGUMENTS에서 URL과 카테고리를 파싱하세요:
- 첫 번째 인자: URL
- 나머지 인자: 카테고리 (쉼표 또는 공백 구분)

카테고리가 없으면 AskUserQuestion으로 질문하세요.

WebFetch로 URL 내용을 읽고, 요약/키워드/인사이트를 생성한 후 `save_article` tool을 호출하세요.

저장 완료 후 결과를 간단히 보여주세요:
- 제목
- 카테고리
- 요약 (생성된 것)
- 키워드
