# /ask-articles

저장된 아티클을 기반으로 RAG 질문 답변을 수행합니다.

## 사용법

```
/ask-articles <질문>
```

## 워크플로우

1. **질문 확인**: $ARGUMENTS에서 질문 파싱
2. **관련 아티클 검색**: `search` tool로 관련 아티클 찾기
3. **청크 검색**: `get_relevant_chunks` tool로 관련 청크 가져오기
4. **답변 생성**: 청크 기반으로 답변 작성
5. **출처 표시**: 참조한 아티클 정보 표시

## 실행

$ARGUMENTS가 비어있으면 질문을 요청하세요.

### Step 1: 관련 아티클 검색
`search` tool로 질문과 관련된 아티클 검색 (limit=5)

### Step 2: 관련 청크 검색
`get_relevant_chunks` tool 호출:
- query: 사용자 질문
- limit: 10

### Step 3: 답변 생성
검색된 청크를 기반으로 질문에 답변하세요:
- 청크에 있는 정보만 사용
- 청크에 없는 정보는 "저장된 아티클에서 찾을 수 없습니다"라고 답변
- 한국어로 답변

### Step 4: 출처 표시
답변 끝에 출처 표시:

```
---
**출처:**
- [아티클 제목 1](article_id)
- [아티클 제목 2](article_id)
```

답변할 수 없는 경우 관련 아티클 저장을 제안하세요.
