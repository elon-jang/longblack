# /search-articles

저장된 아티클에서 시맨틱 검색을 수행합니다.

## 사용법

```
/search-articles <검색어> [--category 카테고리]
```

## 워크플로우

1. **검색어 확인**: $ARGUMENTS에서 검색어 파싱
2. **검색 실행**: `search` MCP tool 호출
3. **결과 요약**: 검색 결과를 보기 좋게 정리

## 실행

$ARGUMENTS가 비어있으면 검색어를 요청하세요.

$ARGUMENTS를 파싱하세요:
- `--category` 또는 `-c` 플래그가 있으면 카테고리 필터 적용
- 나머지는 검색어

`search` MCP tool을 호출하고 결과를 다음 형식으로 표시:

```
## 검색 결과: "검색어"

### 1. [제목] (score: 0.xxx)
- 카테고리: ...
- 저자: ...
- 요약: (excerpt 또는 summary)

### 2. ...
```

결과가 없으면 다른 검색어나 카테고리를 제안하세요.
